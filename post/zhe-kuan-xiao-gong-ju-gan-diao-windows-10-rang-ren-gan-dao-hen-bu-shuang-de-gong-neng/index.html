这款小工具干掉「Windows 10」让人感到很不爽的功能

2020-06-02

有部分人不愿意用「Windows 10」主要还是BUG太多了，今天电脑用着好好的，关机前更新个补丁，明天开机就蓝屏、数据丢失啥的，真是让人头疼。

其实在「Windows 10」频繁出BUG前，最开始微软还在开始菜单的磁贴植入广告，通过收集你个人隐私信息来个性化推广相关的软件，卸载了广告磁贴，下次开机又有了，引起了很多用户不满，装回「Windows 7」系统。

后来微软加入了 "偶尔在开始菜单中显示建议" 功能，允许取消广告推荐，但是收集用户隐私的问题还是引起一些用户的担忧。

这款小工具干掉「Windows 10」让人感到很不爽的功能 第2张

默认「Windows 10」会收集软件安装数据、浏览器流量数据、搜索数据、手写、录入、语音数据、设备配置数据等等。

如果你激活了 Cortana 那就默认允许收集下面的信息：

你的设备的位置
你的电子邮件和短信数据
你的日历数据
使用的应用程序
你的联系人列表
谁正在与你通话
你经常与谁联系
你的闹钟设置
你设备上的音乐
你购买了哪些东西
如果你使用了必应（Bing）搜索引擎，还会搜集你的搜索历史

后来在发布「Windows 10」创意者更新时，微软表示搜集的各种用户数据是为了提供更好的体验！但是目前只有BUG越来越多，更好的体验不存在的！如果不想被微软 "监视" 可以关闭「Windows 10」隐私收集。

微软也是很鸡贼，把各个隐私开关功能藏得很深，一般用户很难找得到开关，有些还需要通过注册表来关闭。

所以今天雷锋哥要给大家推荐一款小工具「WPD」可以非常方便的来关闭「Windows 10」隐私收集、关闭自动更新、关闭 Windows Defender 杀毒软件等。

WPD介绍

这款「WPD」工具小巧免费，最新版已经支持了「Windows 10」2004，运行需要 NET Framework 4.5 支持。功能上主要三部分：隐私管理、防火墙设置、应用商店程序卸载。

隐私管理设置：列出来了「Windows 10」所有跟隐私相关的内容，列表雷锋哥数了下，至少有50个隐私相关的项目，每个项目后面还有个?图标，鼠标点击可以查看相关介绍。重度隐私用户，直接拉到最下面，一键禁用全部，重启系统完事。

这款小工具干掉「Windows 10」让人感到很不爽的功能 第4张

防火墙设置：可以关闭 Windows Defender，要知道 Windows Defender 也是 "硬盘杀手" 有时候会导致过高的磁盘占用率，很多人会选择其它杀毒软件来代替它。

还可以阻止微软应用如Skype、OneDrive，以及关闭「Windows 10」自动更新功能。关闭更新可以100%阻止BUG补丁，在系统没啥毛病情况，不要更新补丁了。

另外也会也能阻止微软收集数据的 IP 地址，会把 IP 地址添加到 HOSTS 文件里。

这款小工具干掉「Windows 10」让人感到很不爽的功能 第5张

应用商店程序卸载：可以卸载「Windows 10」自带的应用，如果你用不到应用商店，也可以给删除了。

这款小工具干掉「Windows 10」让人感到很不爽的功能 第6张

总结

「WPD」提供的功能真是恰到好处，都是用来干掉「Windows 10」让不少人感到很不爽的功能。另外如果对隐私方面还存在担心，可以不要用微软账户登录「Windows 10」系统，改用本地账户。
